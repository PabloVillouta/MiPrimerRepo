\section*{Introduction}
    El algoritmo que voy a presentar fue creado por Pablo Villouta. \\
    Este tiene la función de recibir una cantidad X de elementos para devolverlos de manera ordenada.\\
\section*|{}
\section*{El algoritmo}
    La forma en la que funciona el código es que: \\
    Primero, la consola pide ingresar la cantidad de elementos que vas a ordenar. Esto solo funciona si el numero ingresado es mayor que 0. 
    Segundo, el programa pide que escribas todos los elementos que va a tener tu lista. Para este paso no importa el tipo de elemento que uno seleccione
    Tercero, se le pide al usuario la forma en la que va a ordenar la lista, teniendo 3 posibles formas de hacerlo. Estas son de menor a mayor, mayor a menor o aleatorio.\\
    Una vez terminados todos los pasos, el algoritmo le devuelve a la persona que lo esta usando una lista, ordenada según la opción que seleccionó.
